# OpenClaw Brain

Central AI command hub for Spencer's digital empire. OpenClaw orchestrates all agents, tools, and digital assets from one place.

## Architecture

```
OpenClaw (Central Brain)
  |
  ├── base44superagent (Primary Execution Agent - Base44.com)
  |     ├── Insurance Ops Division
  |     ├── Marketing & Growth Division
  |     ├── Finance & Admin Division
  |     └── Tech & Automation Division
  |
  ├── Kim (AI Ops Manager - Telegram)
  |     └── Task queue, memory, web-fetch, model-switch
  |
  └── Future Agents
        └── (Plug in as needed)
```

## Directory Structure

```
openclaw-brain/
├── hub/              # Central orchestration config
├── agents/           # Agent definitions & configs
│   └── base44super/  # Base44 superagent connection
├── divisions/        # Business division configs
├── memory/           # Shared context & knowledge
├── skills/           # Shared skills & tools
└── playbooks/        # Standard operating procedures
```
